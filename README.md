# SettingsRestart
Simply adds a restart button on Settings, next to the Exit button.

### MelonLoader
Need to install MelonLoader?<br>
Click [this link](https://melonwiki.xyz/) to get started!

### Prerequisites
MelonLoader: v0.3.0+ (Alpha)<br>
Game: VRChat build 1102+
